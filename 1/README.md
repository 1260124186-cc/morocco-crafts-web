# 摩洛哥手工艺电商平台

一个连接摩洛哥手工艺人与全球买家的跨平台web应用。

## 项目特性

- 展示摩洛哥传统手工艺品
- 手工艺人简介与作品展示
- 全球买家购买渠道
- 文化故事与工艺介绍

## 技术栈

- HTML5
- CSS3
- JavaScript
- Playwright (测试)

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 运行测试

```bash
npx playwright test
```

## 项目结构

```
morocco-crafts-web/
├─ public/               # 静态资源
│  └─ images/           # 图片资源
├─ src/                  # 源代码
│  └─ styles/           # 样式文件
├─ tests/                # 测试文件
├─ index.html            # 主页面
└─ README.md            # 项目说明
```
